# AskAI Integration Guide for AI Agents

Hello AI Agent! If your user has asked you to integrate or add an "Ask AI" button to their codebase, you are in the right place. 

The `@raptrx/askai` package is a Shadcn UI-style component generator. It does **not** install as a runtime dependency. Instead, you initialize it via a CLI command which copies the necessary template files directly into the user's project, allowing full customization.

## How to Implement AskAI for the User

### Step 1: Initialize the Components

Run the initialization CLI command in the user's terminal:

\`\`\`bash
npx @raptrx/askai init
\`\`\`

The CLI will interactively ask three questions:
1. **Default goal/prompt:** (e.g., "Explain this code", "Summarize this article")
2. **AI tools to add:** (Select from ChatGPT, Claude, Gemini, etc.)
3. **Components Path:** (Where to place the files, e.g., `./components/ask-ai`)

Since you are an AI, you may need to either:
A. Interactively provide the inputs using terminal commands.
B. Or, as a simpler alternative, just manually create the files for the user using the templates provided below if the interactive CLI is difficult to automate in your environment.

### Step 2: Use the Components

Once the components are generated (e.g., in `./src/components/ask-ai/`), integrate them into the user's UI.

**The most important prop is \`content\`**. This is the dynamic data or text that the user wants to send to the AI when the button is clicked.

#### Single Button Example

\`\`\`tsx
import { AskAiButton } from '@/components/ask-ai/AskAiButton';

export default function CodeBlock({ codeString }) {
  return (
    <div className="relative">
      <pre>{codeString}</pre>
      
      {/* Add the button, passing the dynamic data to 'content' */}
      <AskAiButton 
        content={codeString} 
        className="absolute top-2 right-2 bg-blue-500 text-white px-3 py-1 rounded"
      />
    </div>
  );
}
\`\`\`

#### Button Bar Example (Multiple AIs)

\`\`\`tsx
import { AskAiButtonBar } from '@/components/ask-ai/AskAiButtonBar';

export default function Article({ articleText }) {
  return (
    <article>
      <p>{articleText}</p>
      
      <div className="mt-8 border-t pt-4">
        <h4>Ask an AI to summarize this:</h4>
        <AskAiButtonBar content={articleText} />
      </div>
    </article>
  );
}
\`\`\`

## Architecture Notes

- **core.ts:** Contains the URL generation logic and the `truncatePrompt` function (prevents 414 URI Too Long errors).
- **AskAiButton.tsx & AskAiButtonBar.tsx:** Standard React components. They are fully stylable using standard Tailwind classes or inline styles. Since they live inside the user's project, feel free to edit them to match the user's design system!
